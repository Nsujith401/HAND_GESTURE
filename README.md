HAND GESTURE
----------------------

Computer Vision Base: This project uses OpenCV to capture real-time video and process camera frames for a mirror-like experience.

AI Hand Tracking: It leverages MediaPipe's pre-trained Machine Learning models to accurately detect 21 hand landmarks in milliseconds.

Helper Module: The Cvzone library is used to easily extract finger coordinates and calculate the distance between landmarks for gesture detection.

GUI Automation: PyAutoGUI maps the camera coordinates to the monitor's resolution, enabling touchless cursor movement and clicking.

Human-Computer Interaction (HCI): By combining these tools, the system creates a seamless "Invisible Keyboard" and virtual mouse interface without physical hardware.

Controls
----------------

Move Mouse: Move your index finger in front of the camera.

Left Click: Pinch your thumb and index finger together.

Exit: Press 'q' on your keyboard while the camera window is active.
